## 使い方
1. リポジトリをClone
2. 下記コマンドで必要なRuby Gemsをインストール  
```bundle install --without production```
3. データベースへのマイグレーションを実行  
```rails db:migrate```
4. テストを実行して確認  
```rails test```
5. Railsサーバーの立ち上げ  
```rails server```
